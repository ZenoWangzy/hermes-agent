## Profile Override Timing

The `--profile/-p` flag in `main.py` is processed **BEFORE** any hermes module import. This ensures `get_hermes_home()` returns the profile-specific path for all subsequent imports. Never cache `get_hermes_home()` at module level in code that might be imported before profile override.

## Adding a New Subcommand

1. Add `CommandDef` entry in `commands.py` `COMMAND_REGISTRY` list
2. Create handler in `hermes_cli/<command>.py`
3. Register with argparse in `main.py` subparser section
4. Autocomplete, gateway slash commands, and help text derive automatically from `COMMAND_REGISTRY`
