## Module Purpose

Implements the Agent Client Protocol (ACP) server for editor integration. Enables Hermes to work as an AI assistant inside VS Code, Zed, JetBrains, and other editors that support the ACP standard. Communicates via JSON-RPC over stdio.

## Entry Point

- Command: `hermes-acp` or `hermes acp`
- Module: `acp_adapter.entry:main`
- Starts the ACP agent server with logging to stderr (stdout reserved for JSON-RPC transport)
