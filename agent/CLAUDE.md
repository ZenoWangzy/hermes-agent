## Development Notes

### Sync vs Async Boundary

The core agent loop is **synchronous** (OpenAI-spec tool calling). Async is only used for I/O-bound operations:
- Messaging adapters (gateway platforms)
- MCP client connections
- Browser automation
- Memory provider `sync_turn()` calls

When adding code to agent/, default to synchronous. Use `asyncio.run()` or thread pools for async calls from sync context.

### Circular Import Safety

Import chain: `registry.py` -> `tools/*.py` -> `model_tools.py` -> `run_agent.py` -> `agent/*.py`
Never import `run_agent` or `model_tools` from agent/ modules — use late imports or dependency injection.
