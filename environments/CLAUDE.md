## Thread Pool

`agent_loop.py` uses a `ThreadPoolExecutor(max_workers=128)` for dispatching synchronous tool calls. This default handles parallel tool execution in RL training. Adjust only if hitting system resource limits.

## Adding a New Tool Call Parser

1. Create `environments/tool_call_parsers/<model>_parser.py`
2. Implement parser function matching the expected input/output format
3. Register in `agent_loop.py` parser dispatch table
4. Add tests in `tests/environments/`
